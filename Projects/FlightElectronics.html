Flight Electronics

The flight electronics comprise the control system for the rocket. This system is responsible for all of the sensor logging, valve control, ejection charges, and the data downlink.

CPU

The CPU, or central processing unit, is a set of processors responsible for all high level commands and some of the lower level interfacing. This CPU board contains a LPC11U24 and two ATMEGA32P-AU. The LPC11U24 is an ARM processor is the I2C Master and sends all the data requests. It will log data from a gyroscope, accelerometer, manometer, and altimeter. The sensor fusion algorithm will be processes by one of the ATMEGA32P-AU
